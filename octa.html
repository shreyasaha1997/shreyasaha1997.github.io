Visual Matrix Prediction (2022 - Present)

Advisor - Pengtao Xie, Alireza Kamalipour

    Optical coherence tomography angiography (OCTA) images are used by ophthalmologists to provide three dimensional visualizations of the vascular networks in eyes. Using these images, we try to predict the visual field matrix of an individual. Visual Field analysis is used to detect loss of vision due to glaucoma and other optic nerve conditions. There are two types of visual field matrix (24-2 and 10-2).
    We experimented with various neural network architectures attached with a fully connected layer in the end which predicted each element of the visual field matrix. Since each element in the matrix corresponded to a specific part of the eye image, we have used Unets so that the local information about the image is preserved.
    So far the best correlation obtained is 0.72 and the best absolute mean score is 3.01.
